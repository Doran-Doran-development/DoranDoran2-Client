# 도란도란

## 서비스 소개
도란도란은 교내 실습실 예약 및 관리 서비스와 학생 외출 관리 서비가 융합된 종합 관리 서비스 입니다.

## 기능 

### **회의실**
 - 회의실 생성
 - 회의실 관리
 - 회의실 정보 수정

### **예약**
 - 회의실 예약
 - 회의실 사용 팀 생성
 - 회의실 사용 팀 수정 및 삭제

### **교실**
 - 모든 반 인원 수 및 이동사항 확인
 - 자신의 현 상태 표시
 - 자신의 반 현황 표시

### **외출**
 - 외출 신청
 - 외출 신청 목록 확인 및 승인
 - 외출 상태 표시
